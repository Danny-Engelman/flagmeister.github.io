FlagMeister is smart and small

FlagMeister extends the IMG tag with Custom Element is= notation, executes JavaScript to generate SVG and sets the IMG src

  • FlagMeister has a small JavaScript 21 KB footprint because flags have a lot in common:



  • All flags have been checked for correct Flag colors by https://www.schemecolor.com/color/country-flag
  • Flags are NOT true size but drawn on a standardized 640 by 480 grid
  • Flags requiring higher detail (external SVG) are only loaded when needed (if width exceeds presets)
  • Country names included as 'country' data-attribute, no external JSON file required
  • Loads of extra SVG goodies

FlagMeister adds SVG goodies

Draw on flags

The draw attribute/property accepts any valid SVG string or FlagMeister commands and adds it on top of the FlagMeister flag (a 640 x 480 grid)


FlagMeister commands and parameters

  • backgroundcolor b: [color = #fff]
  • circle c: cx , cy , radius , color [, stroke width , strokecolor]
  • rect: x , y , width , height , color , [stroke width , strokecolor]
  • line: width , color , x1 , y1 , x2 , y2 , [stroke width , strokecolor]
  • flag: iso , [stroke width , strokecolor]
  • background: color
  • attr: attribute , fontsize , y OR attr: text-JSON
  • text: JSON Object: (showing default values):
      { str = 'fm' , size = 520 , x = 320 , y = 430 , fill = false , stroke = '' , s = stroke.split`,` , font = 'Bookman Old Style' , weight = 'bold' , anchor = 'middle'}

Notes on drawing

  • FlagMeister is not meant to be a SVG drawing tool, use InkScape for SVG drawing
  • FlagMeister does no error checking, pay attention to single and double quote usage
  • SVG strings can only contain single quotes
  • The FlagMeister text method takes one parameter, a valid JSON string with double quotes
  • Before you do anything with the US flag be sure to read what you are NOT allowed to do with the US flag by US Federal code 36
